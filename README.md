# TikTok App Site

This folder contains a minimal public website for the TikTok developer app used by the VinUniversity marketing dashboard.

## Files

- `index.html`: homepage / app website URL
- `privacy.html`: Privacy Policy URL
- `terms.html`: Terms of Service URL
- `styles.css`: shared styles

## Recommended TikTok form values

After publishing this folder to a public HTTPS host, use:

- `Web/Desktop URL`: `https://<your-site>/`
- `Privacy Policy URL`: `https://<your-site>/privacy.html`
- `Terms of Service URL`: `https://<your-site>/terms.html`

## Quick publish options

### GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this folder to the repository root
3. In repository settings, enable `Pages`
4. Choose the main branch as the source
5. Wait for GitHub to publish the site

### Netlify

1. Create a new Netlify site
2. Drag and drop this folder
3. Wait for Netlify to publish the site

## TikTok verification

If TikTok asks you to verify URL properties:

- Choose `URL prefix` if you only control this published site
- Choose `Domain` only if you control DNS for the whole domain
